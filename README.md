Welcome to I-RECON, an OSINT (Open Source Intelligence) dashboard developed by ICREATIVE CO. This platform curates a set of browser-based scripts ("bookmarklets") for security researchers, privacy analysts, and ethical hackers to extract and interact with publicly available data across major social networks.

All scripts run locally in your browser, don't require credentials, and are designed for responsible, ethical use.

üåü How to Use

    Install Bookmarklet Tools
        Drag the button for a tool into your browser bookmarks bar.
        Go to the target website (e.g., BandLab, Smule, Facebook).
        Click the saved bookmark to run the tool, extracting data from the current page context.

    Edit Scripts
        Click "Edit" to modify code directly in the browser.
        Click "Save" to download your edited script as a .js file.

    Download & Copy
        Use "Copy" to place code on your clipboard for further use.
        Use "Save as .js" to download source code.
